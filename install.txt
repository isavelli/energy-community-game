NOTE: It requires Node.js (https://nodejs.org)

To create the folder and handle all dependencies, run the following commands in the Windows shell (Start -> Command Prompt (or Prompt dei comandi):

npx create-react-app energy-community-game
cd energy-community-game
npm install lucide-react recharts

---
If different, please replace src/App.js with:

import EnergyGame from './EnergyGame';

function App() {
  return (
    <div className="App">
      <EnergyGame />
    </div>
  );
}

export default App;

---

Finally, just double-click on the "start.bat" file to run the game.
