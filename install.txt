NOTE: It requires Node.js (https://nodejs.org)

To create the folder and handle the dependencies, run the following commands:
npx create-react-app energy-community-game
cd energy-community-game
npm install lucide-react recharts

---
If different, please replace src/App.js with:

import EnergyGame from './EnergyGame';

function App() {
  return (
    <div className="App">
      <EnergyGame />
    </div>
  );
}

export default App;

---

Finally, run the following command to start the game:
npm start
